# 飞书机器人配置指南

> 将飞书机器人连接到 OpenClaw，实现消息接收与回复

---

## 1. 飞书开放平台创建应用

### 1.1 创建应用

1. 访问 https://open.feishu.cn/app
2. 点击"创建企业自建应用"
3. 填写应用名称（如 OpenClaw）

### 1.2 获取凭证

创建后进入应用，获取：
- **App ID**（如 `cli_xxxxxxxxxxxxx`）
- **App Secret**

### 1.3 添加权限

在"权限管理"中添加以下权限：
- `im.message:receive_v1` - 接收消息
- `im.chat:chat:readonly` - 获取群聊信息
- `im.chat:members:get_chat_members` - 获取群成员

### 1.4 事件订阅

1. 进入"事件订阅"
2. 点击"创建订阅"
3. 订阅事件：`im.message.receive_v1`
4. 设置回调地址（需要可访问的 HTTPS 地址）

---

## 2. OpenClaw 配置

编辑 `~/.openclaw/openclaw.json`：

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "accounts": {
        "default": {
          "appId": "你的App_ID",
          "appSecret": "你的App_Secret",
          "mode": "websocket"
        }
      },
      "groupPolicy": {
        "mode": "allowlist",
        "allowlist": []
      }
    }
  }
}
```

---

## 3. 验证配置

```bash
# 重启 Gateway
systemctl --user restart openclaw-gateway.service

# 查看状态
openclaw status

# 查看飞书连接状态
openclaw status --deep
```

---

## 4. 添加机器人到群聊

1. 在飞书开放平台 → 应用管理 → 你的应用
2. 点击"添加机器人"
3. 选择要添加的群聊
4. 在群聊中 @机器人 测试

---

## 5. 常见问题

### Q1: 机器人不回复消息
- 检查应用权限是否正确添加
- 检查事件订阅是否启用
- 检查 appId 和 appSecret 是否正确
- 查看日志：`openclaw logs --follow | grep feishu`

### Q2: 消息发送失败
- 检查机器人是否在群聊中
- 检查是否有发送消息的权限

### Q3: WebSocket 连接失败
- 确保回调地址可访问
- 检查防火墙是否阻止连接

---

## 6. 安全建议

- 不要泄露 App Secret
- 定期轮换 App Secret
- 限制机器人加入的群聊范围

---

**注意**：本文中的 App ID 和 App Secret 请替换为你的实际值。
